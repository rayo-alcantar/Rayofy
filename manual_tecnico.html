Manual Técnico - Rayofy

1. Descripción General

Rayofy es una aplicación de escritorio desarrollada en Python con wxPython y Spotipy para gestionar playlists de Spotify. Permite buscar, añadir, eliminar y editar playlists y canciones de forma accesible y eficiente.

2. Estructura del Proyecto

  • Rayofy.py: Ventana principal y lógica de la interfaz.
  • PlaylistManager.py: Lógica de gestión de playlists y canciones.
  • SearchFrame.py: Ventana para buscar y añadir canciones.
  • SpotifyAuthenticator.py: Autenticación con la API de Spotify.
  • credentials.json: Credenciales de la API de Spotify.

3. Instalación y Configuración

  1. Instala dependencias: 
    pip install wxPython spotipy pyperclip
  2. Crea credentials.json con tus credenciales de Spotify.

4. Uso de la Aplicación

  • Ejecuta Rayofy.py.
  • Navega con el teclado. Usa la tecla de aplicaciones para menús contextuales.
  • Pulsa F5 para refrescar playlists.
  • Usa &Buscar para buscar canciones.

5. Accesibilidad y Usabilidad

  • Todos los controles tienen atajos de teclado y tooltips.
  • El árbol de playlists muestra el número de canciones.
  • Confirmación al eliminar elementos.
